- 매출 **월 200만원** (광고 협찬 / AI 컨설팅 / 상품 / 기타 합산)

**"잘 됐다" 기준 — 바이럴 여부**
- 도달 + 공유 + 저장 + 댓글 + 좋아요 **5종 모두**
- 단일 지표 임계 ❌. 5종 통합 "바이럴 됐는가"가 판단

**1차 데이터 사용자 입력 슬롯**
- ❌ **현재 거부**. 자동화 우선 추진.
- 사유 ↓ (절대 자동화 X 영역과 동일)

**절대 자동화 ❌ 영역**
- 현재 **없음**.
- 조건부 룰: "사용자가 Claude로 직접 디자인 엔지니어링한 카드뉴스 1장과 비교해, 자동화 출력이 동일 급 또는 그 이상 도달하면 자동화 유지·확장. 미달이면 그 영역만 인간 영역으로 분리."
- 즉 **사용자 본인 직접 제작 1장 = anchor benchmark**. 자동화 vs anchor 비교 안 하면 의미 없음.

## 운영 룰 — Anchor 비교 사이클

이 4단계가 본질 anchor가 작동하는 방식.

1. **사용자가 Claude로 카드뉴스 1장 직접 제작** — gold standard
2. 같은 토픽으로 **자동화 파이프라인 1장 출력**
3. 사용자가 1·2를 1:1 비교 — "자동화 ≥ anchor"인가 판정
4. 자동화 < anchor → 어느 영역이 부족한지 정밀 진단 → 룰 보강 → 재시도. 자동화 ≥ anchor → 그 영역 자동화 신뢰 + 다음 영역으로 이동

**anchor 비교 1회 도달 전**: 1차 데이터 입력 슬롯·KPI 누적·brand_voice 자동 학습 등 새 기능 추가는 보류. 양산 정밀도가 anchor에 도달하는 게 우선.

**anchor 비교 1회 도달 후**: 그 시점에서 1년 목표(2만 팔로워·월 200만원·planb 계약)에 필요한 다음 영역 결정.
